== Description ==

Adds each WooCommerce product's short description to shop and category/tag archive pages.

If you have any other suggestions for other information that could be included, please [add your suggestion via GitHub](https://github.com/OM4/woocommerce-short-description-on-archives/issues).

== Installation ==

1. Download the latest stable version of the plugin from https://github.com/OM4/woocommerce-short-description-on-archives/archive/release.zip.
1. Install this plugin via WordPress Dashboard, Plugins, Add New, Upload.
